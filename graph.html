Advanced Graphing

Standard Plots

Enter comma-separated functions of x (e.g., `x^2`, `sin(x)`) or a single function of x,y (e.g., `sin(x)*cos(y)`) for a 3D surface.

Topographic Contour

Projects a 3D surface `z = f(x,y)` onto a 2D plane with contour lines showing elevation.

Vector Field

Plots vectors `F = < P, Q, R>` at points (x,y,z) in 3D space.

Implicit Surface volumetrics

Renders `F(x,y,z) = C` using continuous volumetric isosurfacing.
