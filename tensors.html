Tensor Calculus

Einstein Summation Convention

Any repeated index (one up, one down) is summed over all dimensions. No explicit Σ needed.

Examples in 3D:
aⁱbᵢ = a¹b₁ + a²b₂ + a³b₃ (dot product)
AⁱⱼBʲ = Σⱼ AⁱⱼBʲ (matrix-vector product)
AⁱⱼBʲₖ = Σⱼ AⁱⱼBʲₖ (matrix product)
Aⁱᵢ = Σᵢ Aⁱᵢ = trace(A) (trace is a contraction)
δⁱⱼ = Kronecker delta (identity tensor)
εᵢⱼₖ = Levi-Civita symbol (⁺1, ⁻1, or 0)

Levi-Civita Symbol ε_{ijk}

Metric Tensor gᵢⱼ

The metric tensor defines distances and angles on a manifold. ds² = gᵢⱼ dxⁱ dxʲ.

Christoffel Symbols Γᵏᵢⱼ

Connection coefficients that define parallel transport. Γᵏᵢⱼ = ½gᵏˡ(∂ᵢgⱼˡ + ∂ⱼgᵢˡ − ∂ˡgᵢⱼ)

Geodesic equation: d²xᵏ/dτ² + Γᵏᵢⱼ ·(dxⁱ/dτ)(dxʲ/dτ) = 0
Covariant derivative: ∇ᵢVᵏ = ∂ᵢVᵏ + ΓᵏᵢⱼVʲ
Riemann curvature from Christoffel:
Rˡₖᵢⱼ = ∂ᵢΓˡⱼₖ − ∂ⱼΓˡᵢₖ + ΓˡᵢₘΓᵐⱼₖ − ΓˡⱼₘΓᵐᵢₖ
Γ=0 everywhere ⟺ flat space (in suitable coordinates)
