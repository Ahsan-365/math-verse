Information Theory

Shannon Entropy H(X) = −Σ pᵢ log₂(pᵢ)

Entropy measures average information (surprise) in a probability distribution. Maximum when uniform.

Binary entropy function: H(p) = −p·log₂p − (1−p)·log₂(1−p)
Max entropy: log₂(n) bits for n equally likely outcomes
Self-information: I(xᵢ) = −log₂(pᵢ) bits

Divergences & Mutual Information

KL Divergence: D_KL(P‖Q) = Σ P(x)·log₂(P(x)/Q(x)) ≥ 0
Mutual Information: I(X;Y) = H(X) − H(X|Y) = H(Y) − H(Y|X)
Cross-entropy: H(P,Q) = H(P) + D_KL(P‖Q)
JS Divergence: JSD(P‖Q) = ½D_KL(P‖M) + ½D_KL(Q‖M), M=(P+Q)/2
KL is not symmetric: D_KL(P‖Q) ≠ D_KL(Q‖P) in general.

Channel Capacity — Shannon–Hartley Theorem

C = B·log₂(1 + S/N) bits/second. The maximum rate at which information can be transmitted reliably.
