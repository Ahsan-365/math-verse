Measure Theory & Integration

σ-Algebras & Measurable Spaces

A σ-algebra Σ on set X is a collection of subsets closed under complement and countable unions. (X, Σ) is a measurable space.

σ-algebra axioms:
1. X ∈ Σ (whole space)
2. A ∈ Σ ⟹ Aᶜ ∈ Σ (closed under complement)
3. A₁,A₂,... ∈ Σ ⟹ ∪Aᵢ ∈ Σ (closed under countable union)

Smallest σ-algebras:
Trivial: {∅, X} | Every algebra: {∅, {a}, {b,...}, X, ...}
Borel σ-algebra ℬ(ℝ): generated by open intervals (a,b)
Contains all open, closed, Fσ, Gδ sets. Cannot be explicitly described.

Lebesgue Measure on ℝ

μ([a,b]) = b−a. Extends naturally to countable unions. Outer measure defined via countable interval covers.

Key properties of Lebesgue measure μ:
μ(∅) = 0 | μ([a,b]) = b−a | μ(finite set) = 0 | μ(ℚ∩[0,1]) = 0 (countable!)
σ-additivity: μ(⊔Aᵢ) = Σμ(Aᵢ) for disjoint measurable sets
Translation invariance: μ(A+t) = μ(A)
Cantor set: uncountable, yet has measure 0 (remove middle thirds)
Vitali set: non-measurable set (requires Axiom of Choice)

Lebesgue vs Riemann Integration

Lebesgue partitions the range; Riemann partitions the domain. L integrates more functions and has better convergence theorems.
